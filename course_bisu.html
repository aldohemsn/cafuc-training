课程详情与大纲

北京第二外国语大学高级翻译学院 · 数智化笔译与语料库建设

1 项目背景与培训目标

学员概况

北京第二外国语大学高级翻译学院学员，具备扎实的翻译理论基础和丰富的实践经验。

客户特别要求

  • 增加互动：适当增加互动及提问环节，营造良好课堂氛围
  • 提升参与度：师生人数多，需要调动全员参与
  • 严格时间把控：避免提前结束，准备备用内容
  • 备用方案：提前准备问题或小型案例用于时间调节

培训目标

  • 理解数智化翻译的底层逻辑和发展趋势
  • 掌握实用的翻译技术工具和方法
  • 培养"数据驱动"的翻译思维
  • 建立语料库建设的系统认知

2 90 分钟课程时间轴

模块一：数智化翻译生态概览

0-20 分钟
  • • 开场互动：AI 工具使用情况调查（举手统计）
  • • 讲师介绍 + 课程目标说明
  • • 翻译技术发展脉络（CAT → MT → AI）
  • • 数智化译员能力模型
  • • 互动: 学生分享工具使用痛点（2-3 人）

模块二：高级搜商与术语挖掘

20-45 分钟
  • • Google 高级搜索指令演示（site:, filetype:, 精确匹配）
  • • 实操任务: 查找"碳中和"官方英文表达（3 分钟）
  • • 学生分享搜索结果（3-5 人发言）
  • • 平行文本挖掘策略 + 图像反查验证法
  • • 术语库构建思路
→ 查看搜商构建教程

模块三：AI 辅助翻译实战

45-65 分钟
  • • Prompt Engineering 基础（角色、任务、格式、约束）
  • • 长难句拆解策略演示
  • • 互动实操: 学生设计 Prompt 翻译长难句
  • • 讲师点评典型案例（3-5 个）
  • • AI 质检与术语一致性
→ 查看 AI 辅助教程

模块四：语料库建设入门

65-85 分钟
  • • 语料库概念与价值
  • • 实战案例：《纽约时报》双语库（数据获取 → 清洗 → 对齐 → 检索）
  • • 文本清洗与正则表达式基础
  • • 开放 Q&A: 学生提问与讨论
→ 正则入门 → 语料库教程

课程收尾

85-90 分钟
  • • 核心要点回顾
  • • 资源分享（课件网站、工具清单）
  • • 课后答疑安排

3 互动环节设计

🎤 1. 开场破冰

举手统计、痛点收集

🔍 2. 搜索实操

现场搜索任务 + 结果分享

🤖 3. Prompt 设计

学生设计 + 讲师点评

💬 4. 开放讨论

Q&A + 案例分析
