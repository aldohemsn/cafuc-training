# 翻译技术实战课：CAFUC 定向培训项目介绍

## 1. 项目背景 (Project Context)

### 委托单位
**中国民用航空飞行学院 (CAFUC)** 外国语学院

### 学员画像 (Trainee Persona)
*   **人数/年级**：MTI 研究生一年级（约 20-30 人）
*   **学历背景**：翻译硕士专业学位在读
*   **专业接触**：民航通识英语
*   **已修课程**：
    *   翻译概论、口笔译技巧、翻译理论与文化
    *   译后编辑 (Post-editing)
    *   网络查词技巧
    *   翻译质量评估
*   **技术基础**：接触过 CAT 工具，编程知识基本空白

### 培训目标 (Objectives)
本次培训定位为**"技术意识启蒙 + 理论实践衔接"**：
*   让学生了解翻译技术的发展趋势和底层逻辑
*   与已学的译后编辑、质量评估等课程形成呼应
*   培养"数智化译员"思维，为职业发展奠定基础

## 2. 课程设计 (Curriculum Design)

本课程共 **2 次课**，每次 **90 分钟**。

### 第一次课：从"搜商"到"AI思维"

| 时间段 | 模块 | 内容要点 |
|--------|------|---------|
| 0-15min | 开场引入 | 讲师介绍、课程目标 |
| 15-45min | 术语挖掘与搜商构建 | Google 高级指令、平行文本挖掘、图像验证法 |
| 45-75min | AI 辅助长难句分析 | Prompt Engineering 基础、逻辑拆解、多版本生成 |
| 75-90min | 课堂练习 + Q&A | 使用 Prompt Lab 进行实操 |

> **与已学课程衔接**：网络查词课 → 搜商构建的进阶

### 第二次课：从"清洗"到"质检"

| 时间段 | 模块 | 内容要点 |
|--------|------|---------|
| 0-10min | 回顾与衔接 | 第一次课回顾 |
| 10-40min | 文本清洗与正则入门 | Regex 基础概念、常用模式演示 |
| 40-60min | 翻译质量检测 | AI 质检思路、术语一致性 |
| 60-80min | 语料库建设实战 | 《纽约时报》双语库案例 |
| 80-90min | Q&A | 答疑互动 |

> **与已学课程衔接**：译后编辑 → AI 辅助的底层逻辑；翻译质量评估 → 智能质检的技术实现

## 3. 技术栈 (Tech Stack)

*   **Search**: Google 高级搜索指令
*   **AI Models**: LLM Prompt Engineering (概念介绍)
*   **Text Processing**: 正则表达式 (演示为主)
*   **Data Format**: JSON (概念介绍)

## 4. 交付物 (Deliverables)

*   **交互式课件网站**：基于 Tailwind CSS 构建的静态网站
*   **实操工具**：Regex Lab、Prompt Laboratory
