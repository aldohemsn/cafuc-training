## 核心竞争力：数智化技术流 (The Technical Translator)
与传统翻译讲师不同，Aldo 是一位**"会写代码的翻译家"**。他致力于打破翻译与编程的壁垒，主张通过**Python、正则表达式、AI 模型**等开发者工具，彻底重构传统翻译的工作流。

*   **工具栈**：Linux, Python, PHP, JavaScript, Solr, MySQL, Bash, Sublime Text, Gemini AI。
*   **拒绝平庸**：不使用 Word/Excel 进行低效的人工排重/质检；不依赖黑盒子的商业 CAT 软件。

## 教学理念
*   **技术赋能**：不是教学生"使用工具"，而是帮助他们理解工具背后的逻辑
*   **理论结合实践**：将技术知识与翻译理论（如三角色框架）相结合
*   **启发式教学**：激发学生对技术的兴趣，为其职业发展奠定基础

## 代表性实战案例

### 1. 纽约时报双语数据库 (NYT Bilingual Corpus)
*   **项目描述**：针对《纽约时报》中文网的非结构化新闻数据，构建了一套全流程的自动化采集与检索系统。
*   **技术实现**：
    *   **爬虫**：自动抓取中英双语对照 URL。
    *   **清洗**：将 HTML 解析为结构化的 JSON 格式（含 Metadata, Sentence Alignment）。
    *   **索引**：导入 Apache Solr 搜索引擎。
    *   **前端**：构建基于 PHP/JS 的即时检索界面，支持全文搜索与高亮。
*   **价值**：为译员提供了一个百万级句对的高质量双语平行语料库。

### 2. 智能化航空/ICAO 文档处理流
*   **项目描述**：针对航空制造业（如 COMAC）大量复杂的 PDF 技术规范与工艺文档，开发了一套"清洗-翻译-质检"自动化工作流。
*   **技术实现**：
    *   **清洗**：利用 **Regex (正则表达式)** 瞬间解决 PDF 复制产生的断行、页码混入问题。
    *   **质检**：编写 **Python 脚本**，结合 JSON 术语表，自动扫描供应商译文中的术语不一致与格式错误。
    *   **AI 赋能**：利用 **Gemini 模型** 进行长难句逻辑拆解与风格润色。

### 3. Persona Translation Analyzer
*   **项目描述**：一个基于客户端的 AI 翻译质量评估工具。
*   **特点**：超越传统的字面准确性检查，通过校准的分析框架，诊断译文的风格、结构和细微差别。
